index.html: Interfaz de usuario para la interacción con el sistema.

main.py: Núcleo de lógica y procesamiento de datos.

qr-gorilla.html: Módulo de generación de accesos rápidos (QR) para usuarios.

Creando-embudo-IA-ML.html: Documentación técnica y despliegue del pipeline.

Flujo de Trabajo
Inicialización: Ejecutar main.py para levantar la instancia.

Entrenamiento: Alimentar los modelos mediante el pipeline definido en Creando-embudo-IA-ML.html.

Distribución: Generar accesos vía qr-gorilla.html.
